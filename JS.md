- 声明变量 var

### 数据类型

|  类型  |例子  |
| ------ | --------------- |
| 字符串 | var a = "Hello" |
|  数值      |       var a = 12.34;           |
| 布尔值 | var a = true; |
| 对象 | var a = [123, "ABC"]; |
|  |  |
